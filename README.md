## Description
[]: {{{1

  mdview - markdown preview

  mdview starts a web server that allows you to browse directories and
  view `.md` files.

[]: }}}1

## Examples

    $ mdview        # run on localhost:8888
    $ mdview 9999   # run on localhost:9999
